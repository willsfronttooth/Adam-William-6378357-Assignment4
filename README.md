###Product Performance Analysis 

This repository provides a ready-to-run pipeline that:
- Preprocesses product_sales.csv (handles missing values, outliers, scaling)
- Implements K-means from scratch (kmeans++) and computes elbow & clusters
- Trains Linear and Polynomial regression models to predict profit
- Generates plots and a written REPORT.md (and optionally REPORT.pdf)

Repository structure
